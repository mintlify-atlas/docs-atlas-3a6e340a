`timm` is a library of SOTA computer vision models, layers, utilities, optimizers, schedulers, data-loaders, and augmentations. It provides a consistent, unified API for loading, training, and evaluating any of the 700+ pretrained image models included in the library.

## Get started in three steps

<Steps>
  <Step title="Install timm">
    ```bash
    pip install timm
    ```
  </Step>
  <Step title="Load a pretrained model">
    ```python
    import timm

    model = timm.create_model('efficientnet_b0', pretrained=True)
    model.eval()
    ```
  </Step>
  <Step title="Run inference">
    ```python
    import torch
    import timm

    model = timm.create_model('efficientnet_b0', pretrained=True).eval()
    data_cfg = timm.data.resolve_data_config(model.pretrained_cfg)
    transform = timm.data.create_transform(**data_cfg)

    # Pass a (3, H, W) PIL image through transform, then:
    x = torch.randn(1, 3, 224, 224)  # replace with your transformed image
    with torch.no_grad():
        output = model(x)
    probabilities = torch.nn.functional.softmax(output[0], dim=0)
    ```
  </Step>
</Steps>

<Note>
  timm ships with over 700 pretrained weights. Run `timm.list_models(pretrained=True)` to see all available checkpoints.
</Note>
