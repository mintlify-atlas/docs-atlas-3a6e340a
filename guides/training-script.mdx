## Training script

The training script (`train.py`) evolved from the [PyTorch ImageNet examples](https://github.com/pytorch/examples) and includes significant enhancements such as CUDA-specific performance improvements, native AMP, and distributed training via `distributed_train.sh`.

The dataset folder should be the **base directory** containing `train/` and `validation/` subdirectories.

### Distributed training

For multi-GPU training, use `distributed_train.sh`. The first argument is the number of GPUs.

```bash
./distributed_train.sh 4 \
  --data-dir /data/imagenet \
  --model seresnet34 \
  --sched cosine \
  --epochs 150 \
  --warmup-epochs 5 \
  --lr 0.4 \
  --reprob 0.5 \
  --remode pixel \
  --batch-size 256 \
  --amp \
  -j 4
```

<Tip>
Use PyTorch 1.9+ with native AMP and DDP. `--amp` defaults to native AMP as of timm v0.4.3. Use `--apex-amp` to force NVIDIA Apex components if they are installed.
</Tip>

### Key training arguments

| Argument | Description |
|----------|-------------|
| `--data-dir DIR` | Path to dataset root (must contain `train/` and `validation/`) |
| `--model NAME` | Model name (e.g. `resnet50`, `efficientnet_b2`) |
| `--epochs N` | Total training epochs |
| `--batch-size N` | Per-GPU batch size |
| `--lr LR` | Base learning rate |
| `--warmup-epochs N` | Linear warmup epochs |
| `--sched NAME` | LR scheduler (`cosine`, `step`, `tanh`, `plateau`) |
| `--opt NAME` | Optimizer (`sgd`, `adam`, `adamw`, `rmsproptf`, `lamb`) |
| `--weight-decay WD` | Weight decay |
| `--drop RATE` | Dropout rate |
| `--drop-path RATE` | Stochastic depth drop rate |
| `--aa CONFIG` | Auto/RandAugment config string (e.g. `rand-m9-mstd0.5`) |
| `--reprob PROB` | Random erasing probability |
| `--remode MODE` | Random erasing fill mode (`const`, `rand`, `pixel`) |
| `--model-ema` | Enable model EMA |
| `--model-ema-decay D` | EMA decay factor |
| `--amp` | Enable automatic mixed precision (native AMP) |
| `--dist-bn MODE` | Distributed BN mode (`reduce`, `broadcast`) |
| `-j N` | Number of data loading workers |
| `-c FILE` | Path to a YAML config file |

## Validation script

The validation script (`validate.py`) takes the **validation folder directly** (not the base directory).

```bash
# Validate using a model's pretrained weights
python validate.py \
  --data-dir /imagenet/validation/ \
  --model seresnext26_32x4d \
  --pretrained
```

---

## Inference script

The inference script (`inference.py`) loads a checkpoint and writes top-k class IDs to a CSV.

```bash
python inference.py \
  --data-dir /imagenet/validation/ \
  --model mobilenetv3_large_100 \
  --checkpoint ./output/train/model_best.pth.tar
```

  <Accordion title="EfficientNet-B2 with RandAugment — 80.4 top-1, 95.1 top-5">
    Trained on two Titan RTX GPUs with NVIDIA Apex:

    ```bash
    ./distributed_train.sh 2 \
      --data-dir /imagenet/ \
      --model efficientnet_b2 \
      -b 128 \
      --sched step \
      --epochs 450 \
      --decay-epochs 2.4 \
      --decay-rate .97 \
      --opt rmsproptf \
      --opt-eps .001 \
      -j 8 \
      --warmup-lr 1e-6 \
      --weight-decay 1e-5 \
      --drop 0.3 \
      --drop-path 0.2 \
      --model-ema \
      --model-ema-decay 0.9999 \
      --aa rand-m9-mstd0.5 \
      --remode pixel \
      --reprob 0.2 \
      --amp \
      --lr .016
    ```
  </Accordion>
